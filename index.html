simpleResize

"A simple plugin for resizing"

View on GitHub Download .zip Download .tar.gz

Documentation

1. minWidth
jsFiddle

You can define a minimum width for the td's

var resize = new simpleResize(**table id**,{
       "minWidth": "300px",
});
2. maxWidth
jsFiddle

You can define a maximum width for the td's

var resize = new simpleResize(**table id**,{
       "maxWidth": "500px",
});
3. cursor
jsFiddle

You can define the cursor style

var resize = new simpleResize(**table id**,{
       "cursor": "pointer",
});
4. hover
jsFiddle

You can define the hover style

var resize = new simpleResize(**table id**,{
       "cursor": "red 5px",
});
